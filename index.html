What is WalletBuddy?

WalletBuddy is a simple, fast and easy to use budgeting app. Keep track of your finances by splitting your transactions into categories. The best part? WalletBuddy doesn't require or track any of your personal information in order to track your budget. Simply set a budget and add transactions as you go!

How does it work?

Users will create an account with WalletBuddy. After the account is set up, the user will set their budget and add categories.

The budget is split between several categories which will allow users to add transactions to the correct area. WalletBuddy keeps track of how much money is left on the budget and how much money you've spent on each category!
